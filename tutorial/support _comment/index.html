Giscus comment#

We use the github app giscus to make the comment function works.

You need to install this app to your username.github.io repository, and than configure it in giscus app page, enter your io repository, and choose a mapping rule(we suggest the Discussion title contains page <title> mapping rule).

Copy the generated script, copy it in layouts/partials/comment.html.
